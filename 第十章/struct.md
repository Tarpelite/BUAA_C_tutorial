#结构体

##理解
结构体的整体和成员。
结构体的整体可以和其他变量一样，具体是指：

1. 作为函数的参数
2. 赋值
3. 允许初始化
4. 可以定义对应的指针

但是这些共性之间又有一些不同之处，我们一一探讨。

### 参数传递
结构体在作为**参数传递和声明变量**时特别要注意在类型名包括`struct`

```c
struct student{
	int gpa;
	int grade;
};

void print_student( struct student now){
	printf("gpa=%d grade=%d\n",now.gpa,now.grade);
	return ;
}
```

###赋值

注意！只有相同类型的结构体变量才能互相赋值！！！相同的类型指的是声明时的类型名**完全相同**。
